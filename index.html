RhythmRide

MP3 Player App

The purpose of this MP3 player project is to create a simple, asthetic, and user-friendly music player. It aims to provide an easy way to play, pause, skip tracks, adjust volume, and view playlists, enhancing the overall music listening experience.

Tech Stack:- HTML, CSS and JAVACRIPT

WeatherWise

Weather App

The purpose of this weather project is to integrate API calls into the app design to dynamically display real-time weather data. Additionally, the video background will change to snow, rain, or sunny scenes based on the weather temperature and location (city or country). It also includes dynamic quotes that change with the weather. This enhances the user experience by providing valuable and visually engaging weather information.

Tech Stack:- HTML, CSS and JAVASCRIPT

Shinpi Shop

Watch Ecommerce Store

The purpose of this e-commerce watch website is to offer an engaging shopping experience with user-friendly features like an add-to-cart function, wishlist, advanced search tools, detailed product information, and a price filter. Additionally, it includes seamless Stripe integration for a convenient and secure checkout process.

Tech Stack:- HTML, CSS, JAVASCRIPT, JQUERY and STRIPEJS

Tomo Shop

Clothing Ecommerce Store

The purpose of this online clothing store website is to provide a detailed product catalog with categories, individual product pages featuring descriptions, colors, sizes, and prices. It includes a search bar for quick item lookup, a virtual shopping cart, and filters and sorting tools to help customers easily find products by category, color, and size, ensuring a convenient shopping experience.

Tech Stack:- React, Redux, and SCSS
